# [X岛](https://www.nmbxd.com)养鸽器

### 使用java编写，仅完成主体功能，有概率闪退

## 已知BUG：

 - 使用一段时间后有概率无限闪退，无法再次进入应用（因技术原因无法查明具体错误，现已尽量尝试修复）

 - 删除时有概率删除不应删除的串

 - 删除所有串后保存仍会显示一个串号为0的串

 - 修改串号后新回复可能计算为负数

## 功能：

 - 1.跟踪串回复数，更新时通知

 - 2.选择是否只看Po

 - 3.点击通知跳转到应用界面并将相应通知设为已读

 - 4.导入饼干以访问部分板块
